About Mind Arena

A comprehensive psychological mind games web application designed to challenge and entertain.

Games Included

  • Memory Game - Test your short-term recall with neural network visuals
  • Mind Reader - Mathematical mentalism trick with mystic brain effects
  • Reaction Test - Measure your reflexes with lightning brain animations
  • Personality Quiz - Discover if you're an extrovert or introvert

Technologies Used

HTML5 CSS3 Bootstrap 5 JavaScript Canvas API Canvas Confetti Font Awesome

Project Purpose

This is a Final Semester Project demonstrating interactive web development with psychological themes. The application tests and improves cognitive abilities including memory, attention, reaction speed, and personality traits.

Premium Brain Features

  • 🧠 Animated Brain Neural Networks - Dynamic neuron connections
  • 🔮 Mystic Brain Patterns - Crystal ball and psychic effects
  • ⚡ Lightning Brain Synapses - Fast-paced neural reactions
  • 📊 Psychological Profile Patterns - Personality visualization
  • 🔊 Immersive Audio - Start, click, and win sound effects
  • ✨ Celebration Confetti - Victory explosions
